PassChecker is a lightweight, Python‑based password analysis tool designed to help you quickly evaluate the strength of any password.  
It can **score**, **grade**, and **analyze** passwords — and it can also **generate secure**, **random**, or **disposable** passwords for everyday use.

To get started, run:

```
passchk --help
```

This displays the full list of commands, generators, and usage examples.
